## 主要功能

- **基本資料 / 五大章節填寫**：基本資料、健康管理服務紀錄、測量數據趨勢、健康管理目標、關懷叮嚀、衛教資訊。
- **自動計算次數**：填寫關懷／諮詢日期時，自動統計已完成次數。
- **趨勢圖／衛教圖上傳**：可上傳血壓、血糖趨勢圖與衛教圖片。
- **紀錄管理**：可建立多筆個案，存在瀏覽器中。
- **⧉ 複製本筆**：一鍵把目前個案複製為下一季，快速產生新一季報告。
- **備份 / 還原**：匯出、匯入 JSON 檔，跨電腦或跨瀏覽器搬移資料。
- **報告輸出**：固定 5 頁 A4 制式報告，青綠配色、長輩友善的深黑加粗字體。

## 輸出 PDF 的正確設定

1. 按「👁 預覽」確認版面無誤。
2. 按「🖨 輸出PDF」或直接 `Ctrl + P`。
3. 列印設定：
   - 目的地／印表機：**另存為 PDF**
   - 紙張大小：**A4**
   - 邊界：**預設**（檔案已內含 12mm 邊距，請勿選「無」）
   - 縮放：**100%**
   - **勾選「背景圖形」**（否則框線與底色不會印出）

## 資料保存與注意事項

- 個案填寫的資料是**存在各自電腦的瀏覽器**裡，不會自動同步到其他電腦或瀏覽器。
- 要搬移或交接資料：在來源端按 **📤 備份** 匯出 JSON，到另一端按 **📥 還原** 匯入。
- 清除瀏覽器資料可能導致紀錄消失，**請養成每次填完按「📤 備份」留存 JSON** 的習慣。
- 圖片（趨勢圖、衛教圖）若體積過大可能無法存進瀏覽器，但會包含在 JSON 備份中，因此備份是最完整的保存方式。
